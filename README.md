# Password-Generator

A customizable Password Generator built with React. This project marks my first experience with React, where I learned to manage state, props, and dynamic rendering while building something practical.


✨ Features
- Set Length: Choose how many characters your password should have
- Include Numbers: Toggle whether numbers are part of the password
- Unique Characters: Decide if special characters should be included for stronger security
- Responsive UI: Clean and simple interface that works across devices

🎯 Learning Highlights
- Practiced using React hooks like useState, useEffects, useRef
- Explored conditional rendering and user input handling
